# reports/ — Báo cáo tự động

Báo cáo sáng (PB-4) chạy lúc **05:00 ICT (22:00 UTC)** mỗi ngày qua
`.github/workflows/daily-report.yml`, đổ vào đây dưới dạng `YYYY-MM-DD.md`.

## Template báo cáo sáng
```
# Morning Brief — Swap USD/VND — [YYYY-MM-DD]

## Có gì đổi qua đêm
- LNH O/N: [mới] (Δ vs phiên trước)
- Swap points 1M/3M/6M: [...]
- SOFR / DXY / CNY: [...]
- OMO / tín phiếu SBV: [bơm/hút ròng bao nhiêu]

## Curve hôm nay
- Hình dạng vs hôm qua: [steepen/flatten/không đổi]
- Điểm đáng chú ý: [...]

## 1–2 insight nóng
- [...]

## 🔴 Cảnh báo (nếu có chỉ số chạm ngưỡng watchlist)
- [...]

## Sự kiện trong 48h (từ calendar)
- [...]

## Độ tin & freshness
- [data nào đủ mới / chỗ nào thiếu]
```
